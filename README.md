# Food Price Prediction using Linear Regression

This project demonstrates basic data analysis and machine learning using Python.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Description
The project explores food product datasets and applies linear regression models to analyze relationships between production cost and product price.

## Features
- Data cleaning and preprocessing
- Feature encoding
- Correlation analysis
- Linear regression model training
- Model evaluation (R² score, MSE)
